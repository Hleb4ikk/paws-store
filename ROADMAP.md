# 🐾 Роадмап разработки PawsStore

Подробный план создания интернет-магазина товаров для домашних животных с нуля.

---

## 📋 Этап 0: Подготовка и настройка проекта

### 0.1 Инициализация проекта
- [ ] Создать проект с Vite + React
  ```bash
  npm create vite@latest pawsstore -- --template react
  ```
- [ ] Установить зависимости
- [ ] Настроить структуру папок (`src/components`, `src/pages`, `src/context`, `src/data`)
- [ ] Создать `.gitignore` и инициализировать Git

### 0.2 Настройка Tailwind CSS
- [ ] Установить Tailwind CSS, PostCSS, Autoprefixer
  ```bash
  npm install -D tailwindcss postcss autoprefixer
  npx tailwindcss init -p
  ```
- [ ] Настроить `tailwind.config.js` (content paths)
- [ ] Добавить директивы Tailwind в `src/index.css`
- [ ] Подключить Google Fonts (Inter) в `index.html`

### 0.3 Установка библиотек
- [ ] React Router DOM v7: `npm install react-router-dom`
- [ ] Lucide React (иконки): `npm install lucide-react`
- [ ] React Slick (карусель): `npm install react-slick slick-carousel`
- [ ] Radix UI Select: `npm install @radix-ui/react-select`

### 0.4 Настройка ESLint
- [ ] Настроить правила ESLint в `.eslintrc.cjs`
- [ ] Добавить скрипт `lint` в `package.json`

**Время:** 1-2 часа  
**Результат:** Готовая среда разработки с настроенными инструментами

## 💡 Рекомендации по разработке

### Порядок работы
1. **Сначала функциональность, потом стилизация** — создайте рабочий прототип, затем полируйте UI
2. **Компонентный подход** — разбивайте сложные элементы на переиспользуемые компоненты
3. **Mobile-first** — начинайте с мобильной версии, затем адаптируйте под desktop
4. **Тестируйте часто** — проверяйте каждую фичу сразу после реализации

### Инструменты для ускорения
- **React DevTools** — для отладки компонентов и состояния
- **Tailwind CSS IntelliSense** — автодополнение классов в VS Code
- **Vite HMR** — мгновенное обновление при изменениях
- **Browser DevTools** — для проверки адаптивности

### Частые ошибки
- ❌ Забыть обернуть App в CartProvider
- ❌ Не добавить `key` в списках
- ❌ Не обработать случай "товар не найден"
- ❌ Забыть `e.preventDefault()` при добавлении в корзину
