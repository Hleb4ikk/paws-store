## Технические детали

### Состояние компонента
```javascript
const { id } = useParams();                    // ID из URL
const product = products.find(p => p.id === Number(id)); // Товар
const { addToCart } = useCart();               // Функция из контекста
const [qty, setQty] = useState(1);             // Количество
const [added, setAdded] = useState(false);     // Анимация
```

### Настройки карусели
```javascript
const sliderSettings = {
  dots: true,                              // Точки навигации
  infinite: product.images.length > 1,     // Бесконечная прокрутка
  speed: 400,                              // Скорость анимации
  slidesToShow: 1,                         // Показывать 1 слайд
  slidesToScroll: 1,                       // Прокручивать по 1
  arrows: true,                            // Стрелки
  className: 'product-slider',             // CSS класс
};
```

### Логика добавления в корзину
```javascript
const handleAdd = () => {
  addToCart(product.id, qty);  // Добавить с количеством
  setAdded(true);              // Показать анимацию
  setTimeout(() => setAdded(false), 1500); // Вернуть через 1.5 сек
};
```

### Навигация между товарами
```javascript
const idx = products.findIndex(p => p.id === product.id);
const prev = products[idx - 1];  // Предыдущий товар
const next = products[idx + 1];  // Следующий товар
```
